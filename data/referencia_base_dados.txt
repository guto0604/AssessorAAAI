Referencia Base de dados para LLM
Base de Dados de Assessoria de Investimentos

Este documento descreve os campos das tabelas Informacoes_Cliente, Investimentos_Cliente e Produtos, além de listar os valores possíveis para campos categóricos.

O objetivo é permitir que um modelo de linguagem compreenda a estrutura da base de dados e consiga responder perguntas, gerar insights e interpretar corretamente os dados.

------------------------------------------------
REGRAS GERAIS
------------------------------------------------

Cliente_ID é a chave principal do cliente e conecta as tabelas Informacoes_Cliente e Investimentos_Cliente.

Produto em Investimentos_Cliente representa o nome do produto que o cliente possui em carteira.

Nome_Produto em Produtos representa o catálogo ou prateleira de produtos disponíveis.

Valores monetários estão em reais (BRL).

Datas podem ser usadas para calcular idade, tempo como cliente, dias desde último contato, prazo até vencimento e outros indicadores.

Alguns campos são originais da base e outros são derivados ou sintéticos, criados para enriquecer análises de relacionamento, suitability e oportunidade comercial.
